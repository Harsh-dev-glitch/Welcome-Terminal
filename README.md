# 👨‍🍳 Chef's Terminal

Welcome to the **Chef's Terminal**, my first official project for **Flavourtown 2026**! 

I built this developer dashboard to track my progress and keep my favorite coding tools in one place while I work toward earning a **3D Printer**.

## 🚀 Features
- **The Kitchen Header:** A clean, hacker-style title showing my status as "Cooking."
- **Quick-Access Links:** Direct buttons to Slack, GitHub, and Hack Club Workshops.
- **Responsive Design:** A centered, bordered container with custom CSS glow effects.
- **Live Clock:** A real-time digital clock built with JavaScript (the "heartbeat" of the terminal).

## 🛠️ Tech Stack
- **HTML5:** The skeleton of the dashboard.
- **CSS3:** Custom styling with a dark "Midnight" theme and neon accents.
- **JavaScript:** Logic to power the live time display.
- **Hackatime:** Used to track every minute of the "cooking" process.
